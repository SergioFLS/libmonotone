[MONTONE](https://github.com/MobyGamer/MONOTONE) tracker playback library in C.

# Usage

See [`main.c`](./main.c) for a command-line program that renders a MONOTONE song to a unsigned 8-bit stereo PCM file at 44.1kHz.
